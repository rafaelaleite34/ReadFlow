# ReadFlow
Projeto de um website desenvolvido para organizar e acompanhar meu hábito de leitura. A plataforma permite registrar livros lidos em uma estante virtual, escrever resenhas e visualizar dashboards com a frequência de leitura ao longo do tempo. O objetivo é facilitar a organização das leituras e incentivar a constância no hábito de ler.
